詳解

デバイスプラグイン用のファイル管理クラス。

デバイスプラグイン専用ディレクトリを用意し、そのディレクトリに簡易にアクセスするためのAPIを提供する。

メソッド詳解

- (NSString *) createFileForPath: (NSString *)  path
contents: (NSData *)  contents 

ファイルをデバイスプラグイン専用ディレクトリ以下に作成する。

指定されたパスが絶対パスの場合には、デバイスプラグイン専用ディレクトリではなく、指定されたパスに保存する。

相対パスの場合には、デバイスプラグイン専用ディレクトリにファイルを保存する。

引数
[in]path作成するファイルのパス
[in]contents作成するファイルのデータ
戻り値
保存したファイルへのURL。
nilエラーが生じた場合。

+ (instancetype) fileManagerForPlugin: (DConnectDevicePlugin *)  plugin

デバイスプラグイン向けファイル管理オブジェクトを生成する。

デバイスプラグイン専用ディレクトリ（アプリSandbox内のApplication Support以下のデバイスプラグインと同じ名称のディレクトリ） が存在しない場合、ここで作成する。

+ (NSString *) mimeTypeForArbitraryData

任意のデータ型ファイルのMIMEタイプを返す。

現状、application/octet-streamを返す。拡張子からMIMEタイプを推定できなかった場合、これを代わりに用いるとよい。

戻り値
MIMEタイプ「application/octet-stream」
