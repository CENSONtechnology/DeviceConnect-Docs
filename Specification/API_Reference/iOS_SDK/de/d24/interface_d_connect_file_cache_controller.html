詳解

イベントデータをファイルに保持し、キャッシュの操作機能を提供する。

当クラスのインスタンスを DConnectEventManager::setController: で設定することで イベントデータをファイルで管理することが出来る。 常時はメモリに保持され、DConnectEventManager::flush を実行することでファイルに保存される。 データを永続化する場合は必ずフラッシュすること。 また、初期化時にフラグを設定することでデータ操作時に自動的にフラッシュすることも出来る。
