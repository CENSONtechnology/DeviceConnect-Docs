Detailed Description

Vibrationプロファイル。

Vibration Profileの各APIへのリクエストを受信する。 受信したリクエストは各API毎にデリゲートに通知される。

Method Documentation

- (NSArray *) parsePattern: (NSString *)  pattern

鳴動パターンを文字列から解析し、数値の配列に変換する。

数値の前後の半角のスペースは無視される。その他の半角、全角のスペースは不正なフォーマットとして扱われる。

Property Documentation

- (long long) maxVibrationTime
readwritenonatomicassign

バイブレーションの最大鳴動時間。

バイブレーションのパターンが省略された場合、自動的にデフォルト値として適用される。
デバイスごとに適切な数値を設定すること。 デフォルトでは DConnectVibrationProfileDefaultMaxVibrationTime が設定される。
