詳解

イベントデータをメモリに保持し、キャッシュの操作機能を提供する。

当クラスのインスタンスを DConnectEventManager::setController: で設定することで イベントデータをメモリで管理することが出来る。

注意
インスタンスの破棄と共にイベントデータも破棄されるので注意が必要。
