Detailed Description

イベントデータのキャッシュ操作機能を定義するプロトコル。

イベントデータのキャッシュ操作を行う機能を定義する。 データの追加、削除、検索を提供する。 当クラスの実装クラスのオブジェクトが DConnectEventManager に設定されることで 抽象化されたイベントデータのキャッシュ操作が実行される。
