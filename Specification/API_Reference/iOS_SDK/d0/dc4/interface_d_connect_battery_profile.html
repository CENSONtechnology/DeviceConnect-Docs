詳解

Batteryプロファイル。

Battery Profileの各APIへのリクエストを受信する。 受信したリクエストは各API毎にデリゲートに通知される。

+ (void) setLevel: (double)  level
target: (DConnectMessage *)  message 

メッセージにバッテリーレベルを設定する。

バッテリーレベルは、0.0〜1.0の範囲になる。

  • 0.0の場合は残量なし。
  • 1.0の場合はフル充電。
引数
[in]levelバッテリーレベル(0〜1.0)
[in,out]messageバッテリーレベルを格納するメッセージ
